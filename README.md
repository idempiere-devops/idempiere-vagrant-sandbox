# iDempiere Vagrant Sandbox

An example of how to provision iDempiere with vagrant.

## Common Commands

Start virtual machines:

```shell
vagrant up
```

Stop:

```shell
vagrant halt
```

Restart:

```shell
vagrant reload idempiere
```

Show ports:

```shell
vagrant port idempiere
```

Open terminal:

```shell
vagrant ssh idempiere
```

Destroy virtual machines:

```shell
vagrant destroy
```
